# Retail Sales Analysis Using SQL

## Project Overview

This project focuses on analyzing retail sales data using SQL and MySQL. The objective is to explore customer behavior, sales performance, product categories, and transaction trends by solving real-world business questions.

The project demonstrates the complete SQL analysis workflow, including database creation, data cleaning, data exploration, and business-focused analysis.

---

## Problem Statement

Retail businesses generate large volumes of transactional data every day. Analyzing this data helps organizations understand customer purchasing behavior, identify top-performing categories, improve sales strategies, and make data-driven decisions.

This project aims to answer key business questions using SQL queries and derive actionable insights from retail sales transactions.

---

## Dataset Information

The dataset contains retail transaction records with the following attributes:

| Column Name        |        Description             |
|------------        |-------------                  |
| transaction_id     | Unique transaction identifier |
| sale_date          | Date of sale                  |
| sale_time          | Time of sale                  |
| customer_id        | Unique customer identifier    |
| gender             | Customer gender               |
| age                | Customer age                  |
| category           | Product category              |
| quantity           | Quantity purchased            |
| price_per_unit     | Price per unit                |
| cogs               | Cost of goods sold            |
| total_sale         | Total sale amount             |

## Project Workflow

### 1. Database Creation
- Created Retail Database
- Designed Retail Sales Table
- Imported transaction data into MySQL

### 2. Data Cleaning
- Renamed incorrect column names
- Checked for NULL values
- Verified data quality
- Removed inconsistencies from the dataset

### 3. Data Exploration
- Total number of sales transactions
- Total unique customers
- Product categories available

### 4. Business Analysis

The following business questions were solved:

### Q1. Sales on a Specific Date
Retrieve all sales made on a particular date.

### Q2. Clothing Transactions Analysis
Find Clothing category transactions with quantity greater than 4 during November 2022.

### Q3. Category-wise Revenue Analysis
Calculate total sales generated by each category.

### Q4. Customer Demographics
Find the average age of customers purchasing Beauty products.

### Q5. High-Value Transactions
Identify transactions with sales greater than 1000.

### Q6. Gender-wise Transaction Analysis
Calculate total transactions by gender within each category.

### Q7. Monthly Sales Analysis
Analyze monthly sales performance and identify the best-selling months.

### Q8. Top Customers Analysis
Identify the top 5 customers based on total spending.

### Q9. Unique Customer Analysis
Calculate the number of unique customers purchasing from each category.

### Q10. Sales Shift Analysis
Classify transactions into:
- Morning
- Afternoon
- Evening

and analyze order distribution across shifts.
